# DeepSeek AI 助手

## 🌟 核心功能

- 💬 **智能对话**：基于AI模型的实时对话交互
- 💻 **代码支持**：支持多种编程语言的代码高亮和解析
- 📝 **Markdown渲染**：实时渲染Markdown内容
- ⚙️ **个性化配置**：可自定义API密钥、模型参数等
- 🌓 **主题切换**：支持深色/浅色主题切换

## 🚀 快速开始

## 🔧 技术架构

### 前端技术栈
- **核心框架**：原生JavaScript (ES6+)
- **UI组件**：自定义组件系统
- **状态管理**：基于状态机的全局状态管理
- **样式方案**：模块化CSS设计

### 核心模块
- **AI交互**：ai-mask.js
- **知识库**：knowledge.js 
- **主题管理**：theme.js
- **设置管理**：settings.js
- **API处理**：api.js

### 主要依赖
- Prism.js - 代码高亮
- Marked.js - Markdown解析
- LocalStorage - 本地数据存储

## ⚙️ 使用指南

1. **基础功能**
   - 在对话框中输入问题或指令
   - 使用Markdown格式化内容
   - 复制生成的代码片段

2. **高级功能**
   - 在设置中配置API密钥
   - 切换不同AI模型
   - 导出对话记录
